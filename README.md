# Name
 
青チャート管理システム。
数2B青チャートの学習記録をつけることができます。
 
 
# Features
 
* 青チャートの問題の演習予定日を設定  
* 青チャートの問題の解答時間を記入  
* 青チャートの問題の解答時間の推移を確認  
* 1週間以内に解いた問題・1ヶ月以上に解いた問題をホーム画面に提示  

することができます。
 
# Requirement
 
* Eclipse 4.6 Neon
* MySQL 5.7.xx
